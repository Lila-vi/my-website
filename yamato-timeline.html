ETERNAL ODYSSEY CHRONICLE

宇宙戦艦ヤマト

GREAT VOYAGE — TIMELINE

1974年、この地球に一つのSF作品が生まれた。 それは今や、世界中でSF作品を代表する一つの作品となり、語り継がれている。 かのスターウォーズの監督「ジョージ・ルーカス」も絶賛したといわれる作品。 それが「宇宙戦艦ヤマト」。日本の誇るSF作品の一つである。

作品・放送情報
劇中年 / 劇中事象
1974
宇宙戦艦ヤマト（第1作）
TV / 読売テレビ / 26話
1974年10月6日放送開始。当初はテレビ漫画という時代もあり、視聴率低迷で打ち切り。しかし再放送で爆発的人気を獲得し、日本SFアニメの歴史を塗り替えた。
★ 再放送で社会現象へ。敗者から覇者への大逆転
西暦 2199年
ガミラス帝星による遊星爆弾攻撃。地球は放射能に覆われ、人類滅亡まで1年。絶望に瀕した地球にイスカンダルからコスモクリーナーDと波動エンジンの設計図が届く。戦艦大和を改造した宇宙戦艦ヤマトが、今旅立つ。
1977
劇場版 宇宙戦艦ヤマト
劇場 / 東映 / 編集版
TV版を再編集した劇場版が公開。徹夜の行列、入場制限が続出。興行収入は当時の邦画アニメ記録を更新し「アニメブーム」という言葉を生んだ。
★ スター・ウォーズより前に日本で宇宙SFブームを起こした
西暦 2199年（同上）
無限に広がる大宇宙。静寂な光に満ちた世界。死んでゆく星もあれば、生まれてくる星もある。そうだ、宇宙は生きているのだ。我々の太陽を含む大銀河系もまた、生命に満ち溢れているのだ。しかし、我々の地球は今その最後の終末の時を迎えようとしていた。
1978
さらば宇宙戦艦ヤマト 愛の戦士たち
劇場 / 東映 / 新作
公開直前まで内容が極秘。衝撃のラストシーンは日本中を震撼させた。興行収入21.8億円。当時の子供たちの心に消えない傷を残した名作。
★ 宇宙戦艦ヤマト、地球のために最後の戦いへと赴く。
西暦 2202年
ガミラス戦役が終わり、復興を始めた地球。新鋭戦艦アンドロメダも進宙し、復興の兆しを見せる地球に、一つの通信が入る。そして新たな危機、白色彗星帝国ガトランティスの存在が知らされる。
1979
宇宙戦艦ヤマト2
TV / 日本テレビ / 26話
「さらば」のTV版として制作。ラストが変更され古代進らが生存する結末に。テレサ（テレザート星）との交信と白色彗星帝国との死闘を描く。
★ 「さらば」と同じ物語でラストが違う。ファン間で永遠に語られる二択
西暦 2202年
「さらば」でヤマトは終わらせてはいけない。そういうファンの願いにより復活、再編された奇跡の続編。
1979
宇宙戦艦ヤマト 新たなる旅立ち
TV スペシャル / 日本テレビ / 単発
1979年7月31日放映のTVスペシャル。ヤマト2の後日談として制作。イスカンダルとガミラスの関係、スターシャと古代守の物語、そしてヤマトの新たなる旅立ち。
★ ヤマトシリーズ初のTVスペシャル。スターシャの運命が初めて描かれた
西暦 2202年（ヤマト2後）
ガトランティスの脅威も過ぎ去り平和になった地球。新たな乗組員を迎え入れ、訓練航海中の宇宙戦艦ヤマトにかつての宿敵デスラーから一つの通信が入る。それは正体不明の敵に攻撃されたという、救難信号だった。
1980
ヤマトよ永遠に
劇場 / 東映 / 新作
暗黒星団帝国デザリアムの侵攻。圧倒的技術力に成すすべもなく電撃的に敗北してしまう衝撃的映像。「愛よその日まで」など劇中楽曲が話題を呼んだ。
★ 宮川泰の音楽なしにヤマトは語れない。交響詩として完璧
西暦 2203年
イスカンダルでの事件から数か月後、地球に侵攻してきた帝国、暗黒星団帝国デザリアム。占領された地球を救うためさらなる進化を遂げたヤマトは再び発進する。
1980
宇宙戦艦ヤマト3
TV / 日本テレビ / 25話
1980年10月から放映。ガルマン・ガミラスとボラー連邦の星間大戦に巻き込まれ、太陽の核融合異常増進により再び危機に瀕した地球のため、移民先惑星を求めてヤマトが旅に出る。
★ 太陽の核融合異常増進というイレギュラーの自然現象。地球人類のため、ヤマトは旅立つ
西暦 2203年
原因不明の太陽の核融合異常増進により、残り一年となった地球のため、移民惑星を求め発進するヤマト。しかし、正体不明国家ガルマン、ボラーの戦争に巻き込まれて行ってしまう。
1983
宇宙戦艦ヤマト 完結編
劇場 / 東映 / 新作
回遊惑星アクエリアスが地球に接近し大洪水の危機。自然現象にあらがうためヤマトが出撃。「これがヤマトの最後だ」として制作・公開された。
★ 抗えない宇宙規模の災害。現実と通じるものを感じるとして話題になることもあった。
西暦 2205年
銀河系に突如別銀河が現れ、銀河系全体で災害が発生していた。調査のため派遣された方面でヤマトは、大洪水にあった惑星を発見、一人を救出する。そして回遊惑星アクエリアスの存在が新たな敵とともに示唆されることとなる。
2009
宇宙戦艦ヤマト 復活篇
劇場 / 東映 / 新作
「完結編」から26年ぶりの新作劇場版。西崎義展プロデュースで復活。CGを駆使した宇宙戦闘が見どころ。旧作ファンへの贈り物。
★ 26年ぶりの復活に旧来のファンが涙した
西暦 2220年代
移動性ブラックホールの脅威から人類を救うため、アクエリアスの海に沈んだヤマトが今、復活する。
放送年
作品・放送情報
劇中年 / 劇中事象
2012
宇宙戦艦ヤマト2199
劇場先行 / TV / 26話
2012年4月から劇場上映、2013年4月からTVアニメとして放映。現代の映像美で第1作を完全リメイク。ガミラスの文化・社会構造まで緻密に描いた傑作。
★ オリジナル要素を含めた圧倒的な完成度。
西暦 2199年
人類滅亡まで1年。絶望に瀕した地球にイスカンダルから波動エンジンの設計図が届く。コスモリバースシステムという地球を救うシステムを受け取りに行くため、最新技術で今、ヤマトが再び地球より旅立つ。
2014
宇宙戦艦ヤマト2199 星巡る方舟
劇場 / 新作
2199のオリジナルエピソードとして制作された劇場版。異星人でも分かり合えることを証明するオリジナルストーリー。
★ 2199の世界観をさらに深掘りしたファン必見の補完エピソード
西暦 2199年（イスカンダル帰還途上）
イスカンダル帰還の道中正体不明の敵から攻撃を受けるヤマト、離脱するため緊急ワープを実行したヤマトは濃霧に包まれた空間にワープアウトしてしまう。探査のため出発した一行は降り立った惑星で、同じく迷い込んだガミラス人、そして密林の中に眠るヤマトを見つける。
2017
宇宙戦艦ヤマト2202 愛の戦士たち
劇場 / TV / 26話
「さらば」のリメイクとして制作。テレサとガトランティス帝国の物語を現代映像美で再構築。「コスモリバースシステム」概念が大幅に拡張。
★ 愛と犠牲のテーマをより深く掘り下げた。人が人で生きるため選択する。
西暦 2202年
ガトランティス帝国の侵攻により再び危機に瀕した地球。ガトランティスとの戦いの中、各人物があらゆる「選択」を強いられる。人が人であるための選択を。
2020
宇宙戦艦ヤマト2205 新たなる旅立ち
劇場 / 前後章
旧作「新たなる旅立ち」のリメイク。スターシャの運命、イスカンダルの崩壊、サーシャの登場。デスラー・古代の絆を描く感情的に濃厚な作品。
★ 旧作で書かれなかったガミラス、イスカンダルの関係性が深堀されていく。
西暦 2205年
ガトランティス戦役より復興した地球、新たな乗組員を加えたヤマト、ヒュウガ、アスカの第65護衛隊はイスカンダルへの表敬訪問と訓練航海を兼ねて地球を発進する。
2022
ヤマトよ永遠に REBEL3199
劇場 / 前後章（継続中）
「ヤマトよ永遠に」のリメイク。2022年10月第一章、2024年第二章が公開。暗黒星団帝国の侵攻を現代解釈で描く最新作。
★ リメイクシリーズの集大成へ向けた新章。現在進行形の歴史がここにある
西暦 2207年（予定）
未来の地球を名乗るデザリアム。孤立するヤマト。新たなガルマン・ガミラスの姿。展開される政治戦、プロパガンダ。現実の延長とされるリメイクヤマト、リアリティの高さ故怖さが存在してしまう最新作。
